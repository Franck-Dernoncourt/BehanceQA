# BehanceQA

A Dataset for Identifying Question-Answer Pairs in Video Transcripts 

# Code

To run the rule-based model, use the following commands:

```
cd code/rule
python rule.py
```

To run the BERT-based model, use the following commands:

```
cd code/tranformer
bash my_run.sh
```
